## Phase 5 â€” Server Log Excerpts

### POST /api/bids failure (NOT NULL constraint)
```
error: null value in column "title" of relation "bids" violates not-null constraint
  severity: 'ERROR',
  code: '23502',
  detail: 'Failing row contains (6fbe8acc-..., 5d3b242f-..., null, null, null, null, draft, 0, 0, null, ...)',
  table: 'bids',
  column: 'title'
```
**Root cause:** `POST /api/bids` does not validate required fields before INSERT. If body omits `title`, it passes `null` directly to Postgres which enforces NOT NULL.

### AI SDK Deprecation Warning (non-blocking, repeated on every AI call)
```
AI SDK Warning (deepseek.chat / deepseek-chat): The feature "specificationVersion" is used in a compatibility mode. Using v2 specification compatibility mode. Some features may not be available.
```
**Root cause:** `@ai-sdk/deepseek` v1.0.41 uses a newer spec version than the `ai` SDK v6.0.183 expects. Non-blocking but indicates version skew.
